FAQ

언제 오픈 되는지 궁금해요.
2021년 8월 20일을 목표로 열심히 개발 중입니다 :) 안정적인 서비스로 오픈 할 수 있도록 기다려 주세요!
Dply.app의 요금제는 어떤지 궁금해요.
현재 완벽히 정해지진 않았습니다. 정식 서비스를 기다려 주세요 :(
Dply.app은 어떤 방식으로 호스팅 되는지 궁금해요.
Docker 기반으로 여러 명이 방 처럼 서버를 나누어 호스팅 되는 가상 서버 방식입니다.
이로 인해 서버의 보안을 향상 시키는 작업이 주기적으로 진행될 예정 입니다.
Dply.app은 소셜 로그인(SNS)를 지원하나요?
Dply.app은 아직 Google OAUTH2로만 로그인이 가능하게 개발 중입니다. 추후 여러 SNS를 추가할 예정입니다 :)
가입 할 때 카드 인증이 필요한가요?
저희는 실제 결제 진행시에만 인증을 진행하고 무료 플랜 혹은 가입 만으로는 인증이 필요하지 않습니다.
Dply.app은 클로즈 베타 테스트(CBT)를 진행하지 않나요?
CBT는 이미 모집되었고, 이메일 등으로 개별 안내 예정입니다 :)
Dply.app은 오픈 베타 테스트(OBT)를 진행하지 않나요?
OBT는 현재 오픈 예정 일자부터 이며, 추후 공지시까지 OBT 상태를 유지합니다. :)
